Order Status

Order status

created

The order was created using the booking system but it was not delegated neither paid.

ordered

The order was created using the booking system and it was delegated or paid. Also, the order may had been created by an admin using the BO.

prepaid

The order was paid but no delivered. The previous status may had been created or ordered.

delivered

The order is delivered (done and pdf uploaded) but not paid. The previous status was ordered.

completed

The order is done with pdf uploaded and paid. The previous status was delivered or prepaid.
